# Preface 
- selenium is a set ot tools for automating browsers, offers 
    + selenium IDE, this is a firefox add-on used to record and replay back the selenium scripts with firefox
    
    + selenium webdriver is a programming interface for developing advanced selenium scripts using programming language and offer client libraries 
    
    + selenium standalone server this is known as selenium grid, allows remote and distributed execution of selenium scripts created with webdriver
