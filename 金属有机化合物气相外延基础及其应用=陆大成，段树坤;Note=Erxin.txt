# 前言, 绪论
- MOVPE，金属有机化合物气相外延
- 外延， epitaxy，在...之上排列的意思
- 液相外延， liquid phase epitaxy, LPE, 指在一定去向的单晶衬底上，利用溶质从过饱和溶液中析出生长外延层的技术
- 气相外延， vapor phase epitaxy， VPE
- 氢化物气相外延， HVPE
- 氯化物气相外延，Cl-VPE
- 金属有机化合物气相外延， MOVPE，利用金属化合物进行气相外延
- 分子束外延，molecular beam epitaxy， MBE
- 化学外延，chemical beam epitaxy
- MOVEP概述
    + 三甲基铝，TMAL
    (1-x)(CH_3)_3Ga+x(CH_3)_3Al + AsH_3 = Ga_(1-x)Al_xAs (|v) + 3CH_4
    

# MOVPE生长系统
- 主要组成
材料运输；反应室；尾气处理；控制分系统；原位监测
    + MOVPE(AP-MOVPE)常压
    + LP-MOVPE低压反应室
- 气体运输子系统
氢气，氮气钢瓶，压力调节阀，氢气氮气提纯器，用氮气置换系统中的氢气在维护时
MO源多为液态存储在不锈钢鼓泡瓶中，需要用载气将其带到反应室
    + MO流量估计
    n_MO = F * P_MO / (V_m * (P_bub - P_MO))
    P_MO, 源蒸汽压力
    P_bub，鼓泡瓶内气体压力
    F，通入鼓泡瓶的载气流量，sccm
    V_m，等于22414 (cm^3/mol)
    + 生长/放空多路组合阀
    + 1Torr = 133.32237Pa
    + 双稀释管路， DEZn
    + 单稀释管路
    + 带有超声波浓度计
    + 标准管路
    + EPC， electronic pressure controller，电子压力控制器
- ppb， parts per billion， 体积分数 nl/L
- TBAs, 特丁基砷化氢；TBP，特丁基磷化氢
- 反应室分系统
    + 第一保持反应室内气体无涡流
    + 良好的衬底温度均匀
    + 克服原料沿层所造成的外延层不均匀
    + 防止反应释放的杂质对外延的污染
    + 防止装载片或运行时的空气污染
    + 精确控制反应室内压力
    + 种类
    水平，高速旋转，紧耦合喷淋,  光辅助，等离子
- 两家世界领先公司
AIXTRON， Veeco
- 尾气处理分系统
砷尘粉，白磷冷阱，过滤使用NaBrO_3, NaClO
- 原位测量
    + 发射率校正高温计
    + 反射率计
    + 反射差分光谱，reflectance difference spectroscopy, RDS
    + 表面光吸收，surface photoabsorption， SPA
    + 光束挠度计， deflectometer， 测量外延片的挠曲率
