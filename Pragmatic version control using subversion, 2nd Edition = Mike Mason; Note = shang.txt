The pragmatic starter kit-volume I
Pragmatic version control using subversion, 2nd Edition Mike Mason

Apache subversion系统可以安装在unix下，linux，windows下
有多种行命令，开头语句为svn

是一套版本与仓储管理系统

windows下的图形化管理工具
TortoiseSVN

Eclipse 开源的java ide， 其中插件subclipse来支持subversion http://subclipse.tigris.org

intellij idea另一个流行的java ide

subversion 支持大量的语言绑定，可以通过喜欢的语言访问subversion 的api。语言绑定使用 swig simplified wrapper and interface generator

TMate JavaSVN 库可以从http://tmate.org/svn 下载

帮助的使用 svn help command 即可得到帮助
共同选项
--targets	list	读取list
--non-recurisive, -N 只操作单个目录
--verbose, -v 打印额外的信息
--quiet, -q 打印的信息尽可能地少
--username name 指定在连接授权时使用的用户名
--password
--no-auth-cache 不要缓存身份令牌
--non-interactive 不要提示输入额外的信息
--config-dir dir 从dir读取用户配置信息
--editor-cmd cmd 使用cmd作为日志消息的编辑器
命令
svn add
svn blame以后均已SVN开头
cat, checkout, cleanup, commit, copy, delete, diff, export, import, info, list, lock, log, merge, mkdir, move, propdel, propedit, propget, proplist, propset, resolved, revert, status, switch, unlock, update, 

使用例子
签出
svn checkout url path

签出指定版本
svn checkout -r -rev url
...
