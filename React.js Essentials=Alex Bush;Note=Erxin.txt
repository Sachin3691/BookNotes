# Installing Powerful Tools for Your Project
- 2013 facebook released react, open source on github, 
http://facebook.github.io/react/
https://github.com/facebook/react
- install node.js 
- install git 
- getting data from the twitter 
- filtering data with snapkit engine
    + install snapkit engine 
    $ git clone https://github.com/snapkite/snapkite-engine.git
    
    + install dependent modules for snapkit engine 
    $ npm install --global node-gyp 
    $ cd snapkit-engine/
    $ npm install 

    + install snapkite filters You can find a list of all the available Snapkite Filters at https://github.com/snapkite/snapkite-filters.

- create project structure 
$ npm init 

- build with gulp.js 
Gulp.js. You can learn more about Gulp.js at http://gulpjs.com.

$ npm install --save-dev gulp 
$ npm install --global gulp
